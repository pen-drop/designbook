# Application Shell

## Overview
A minimal header shell for the Keytec agency website. Clean and elegant with maximum focus on content. Logo on the left, navigation links in the center/right area, and a highlighted CTA button for contact.

## Navigation Structure
- Homepage (logo click)
- Services
- Portfolio
- About & Team
- Blog
- Contact (highlighted CTA button)

## User Menu
Not applicable — this is a public-facing Keytec website with no user authentication or login area.

## Layout Pattern
Minimal Header — Logo positioned on the left, navigation links arranged horizontally in the center or right area. The "Contact" item is visually distinguished as a primary CTA button (using the blue primary color). The header is sticky on scroll for persistent navigation access. Content area spans the full width below the header.

## Responsive Behavior
- Desktop: Full horizontal navigation visible with all items, CTA button on the right
- Tablet: Navigation slightly compressed, all items still visible but with reduced spacing
- Mobile: Hamburger menu icon replaces navigation, opens a full-screen or slide-in overlay with all navigation items stacked vertically

## Design Notes
- Use Inter font (heading font) for navigation items for consistency with the design system
- Primary color (blue) for the Contact CTA button and active navigation state
- Neutral color (slate) for header background and text
- Keep the header height minimal to maximize content space
- Consider adding a subtle border or shadow to separate header from content
