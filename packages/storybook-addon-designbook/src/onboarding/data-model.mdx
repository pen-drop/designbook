# Data Model

Define the core entities and relationships in your product. The data model establishes the "nouns" of your system — what things exist and how they connect.
Use the AI command to define or update your data model — the result is displayed here automatically.

<DeboSection
  dataPath="data-model.json"
  parser={(content) => JSON.parse(content)}
  command="/debo-data-model"
  emptyMessage="No data model defined yet"
  renderContent={(data) => <DeboDataModelCard data={data} />}
/>
