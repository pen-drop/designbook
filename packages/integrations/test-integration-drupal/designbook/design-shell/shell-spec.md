# Application Shell

## Overview
The keytec.de shell uses a clean top navigation pattern that reflects the agency's professional, modern design aesthetic. The logo anchors the left side while all navigation elements are right-aligned, creating a clear visual hierarchy and maximum content space below.

## Navigation Structure
- **Logo** (left) → Startseite (Landing Page)
- Leistungen → Services section
- Projekte → Projects / case studies section
- Blog → Blog section
- Agentur → Agency profile section
- Language Switcher (DE/EN)
- CTA Button "Kontakt" → Contact page / form

## User Menu
No user menu required. The right side of the navigation contains the language switcher (DE/EN) and a prominent CTA button "Kontakt" for lead generation.

## Layout Pattern
Top Navigation — Logo on the far left, all navigation items right-aligned. This is the standard pattern for corporate agency websites, projecting professionalism and giving content maximum width for showcasing projects and case studies.

## Responsive Behavior
- Desktop: Logo left, all navigation right-aligned (nav items, language switcher, CTA button) in a single horizontal bar
- Tablet landscape: Same structure as desktop with slightly reduced spacing
- Tablet portrait: Hamburger menu — logo left, hamburger icon right; opens overlay with all nav items, language switcher, and contact CTA
- Mobile: Hamburger menu — logo left, hamburger icon right; opens overlay with all nav items, language switcher, and contact CTA

## Design Notes
- The CTA "Kontakt" button should visually stand out from the regular nav items (e.g., filled/primary button style)
- The language switcher should be subtle and not compete with the main navigation or CTA
- Navigation items map directly to the four roadmap sections: Leistungen, Projekte, Blog, Agentur
- The default view (Startseite) is a landing page with a hero section, not one of the sections
